# Skip-Gram-Model-PyTorch
PyTorch implementation of the word2vec (skip-gram model) and visualization of the trained embeddings using TSNE !

![2D representaion of some of the trained word embeddings](w2v.png)

# Requirements
* torch >= 1.4    
* numpy >= 1.18      
* matplotlib       
* tqdm 
* nltk
* gensim


# Training
```
python main.py
```
